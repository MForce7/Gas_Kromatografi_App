## ⚙️ Installation

### 1. Clone Repository
```
git clone https://github.com/username/project-name.git  
cd project-name
```

### 2. Create Virtual Environment

```
python -m venv venv
```

Activate:

Windows:
```
venv\Scripts\activate
```
Linux / Mac:
```
source venv/bin/activate
```
### 3. Install Dependencies
```
pip install -r requirements.txt
```
